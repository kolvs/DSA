### 时间复杂度分析

|       排序算法       | 时间复杂度 | 是否基于比较 |
| :------------------: | :--------: | :----------: |
|   冒泡，插入，选择   |   O(n²)    |      是      |
| 快排，归并，希尔，堆 |  O(nlogn)  |      是      |
|    桶，计数，基数    |    O(n)    |      否      |



|          | 是否原地排序? | 是否稳定? | 最好    最坏   平均 |
| -------- | ------------- | --------- | ------------------- |
| 冒泡排序 | √             | √         | O(n)   O(n²)  O(n²) |
| 插入排序 | √             | √         | O(n)   O(n²)  O(n²) |
| 选择排序 | √             | ×         | O(n²)  O(n²)  O(n²) |
