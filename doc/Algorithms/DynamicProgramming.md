Dynamic Programming

动态规划比较适合用来**求解最优**问题，比如求最大值、最小值等等。它可以非常显著地降低时间复杂度，提高代码的执行效率。

- DP - 局部最优子结构/多种记录值
- 回溯 - 重复计算
- 贪心 - 永远局部最优
- 递归 + 记忆化 -> 递推
- 状态定义;状态转移方程;
